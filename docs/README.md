# Documentation

This directory contains reference documentation for FortiManager API development.

## Required Documentation (not tracked in git)

Download the following documentation from [Fortinet Developer Network](https://fndn.fortinet.net/):

### JSON API Reference

Download FortiManager JSON API Reference for your version:

```
docs/fndn/7.6.5/json_api_reference/
├── json/           # JSON schema files
├── html/           # HTML documentation
└── swagger/        # OpenAPI/Swagger specs
```

### How-To Guides

Download the FortiManager API How-To Guide:

```
docs/guides/FortiManager API - How-To Guide/
└── howto_jsonrpc_api-main/
    ├── 001_fmg_json_api_introduction.rst
    ├── 008_policy_package_management.rst
    └── ...
```
